# Documentation Index

This directory contains project-level engineering notes and status tracking.

## Files
- `project_analysis.md`: Current project analysis summary and validation snapshot.
- `todo.md`: Active TODO list.
- `constants_guide.md`: Reference for constant definitions and where to edit them.
- `map_format.md`: Legacy + destructible map cell format used by runtime map loading.
- `map_db_format.md`: Binary-only multi-map database format proposal (`NO|TITLE|METADATA` style).
- `asset_composer_format.md`: Draft v1 plan for composed assets (`CELL32`, `SPRITE`, `ANIM`) and `assets.dat`.

## Maintenance rules
- Keep analysis and TODO status aligned with current repository state.
- Prefer linking to source files instead of duplicating implementation details.
- Treat code as source of truth when values in docs and code differ.
